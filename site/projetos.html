Projeto 1

O projeto tem o intuito de promover políticas que reduzem a quantidade de carros que circulam pela cidade. Com isso, utilizar bicicletas para a locomoção de lugares mais próximos. E a utilização de outros combustíveis nos transportes públicos com a finalidade de reduzir os gastos.

Projeto 2

O projeto é um compilado de informações relacionados à mobilidade urbana de forma geral - conceito e planejamento urbano - e sobre a questão regional do município de Nova Friburgo.

Projeto 3

Uma solução para reduzir o custo de uma passagem de ônibus em até 40%, simplesmente reajustando tributação. Um exagero de tributos que atinge apenas a população, esses tributos são responsáveis pelo alto preço das passagens, pois quanto maior o custo do serviço, mais caro ele custa.
